Once `{{AGENT_ID}}.orchet.ai` is live:

```bash

  --manifest-url https://{{AGENT_ID}}.orchet.ai/.well-known/agent.json \
  --bundle ./bundle.tgz \
  --signature-file .orchet/signature.json \
  --contact-email you@example.com
```
