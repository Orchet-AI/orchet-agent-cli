## Local dev

```bash
cp .env.example .env.local          # paste OAuth app credentials
npm install
npm run validate-manifest
npm run dev                         # serves on :3007
```

## Deploy

```bash
vercel link
vercel env add {{CLIENT_ID_ENV}} production
vercel env add {{CLIENT_SECRET_ENV}} production
vercel env add {{AGENT_BASE_URL_ENV}} production    # e.g., https://{{AGENT_ID}}.orchet.ai
vercel deploy --prod
```

Register your {{DISPLAY_NAME}} OAuth app with redirect URI:
```
https://api.orchet.ai/connections/callback
```
